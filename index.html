Sobre

Sou o Itatelino Neto

Formado em Análise e Desenvolvimento de Sistemas, atualmente pós-graduado em Desenvolvimento Web Full Stack. Sou um profissional que gosto de resolver problemas, gosto de desafios complexos. Adepto da cultura Devops, trabalho com versionamento de software usando Git e Github, otimização de ambiente com Docker, Docker Compose e orquesração de containers com Kubernets. Procuro me manter sempre atualizado com dinâmicas otimizadas no desenvolvimento de sistemas robustos e com responsividade, de maneira ágil e dinâmica, garantindo a funcionalidade, segurança e confiabilidade de todo o sistema.

Ferramentas

Tecnologias utilizadas / Minhas Skills.

No back end trabalho com Node.js, Java e PHP com servidor Apache.

Banco de dados com modelos relacionais, uso SQLSERVER e MySQL, a depender da aplicação, uso nosql com MongoDB.

No front end uso HTML como linguagem de marcação de texto, e a estilização com CSS.

Javascript como a liguagem de programçao para o desenvolvimento da interatividade com eventos na aplicação.

Este é um projeto em que, embora seja simples e básico, possui uma responsividade muito bem otimizada.

É possivel visualiza-lo em qualquer dispositivo, seja desktop, tablets ou dispositivos móveis.

O repositório deste projeto está no meu Github, inclusive, este projeto está hospedado no Github!
